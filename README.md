# Titan Fitness Gym

A modern, premium gym landing page built with React + Vite.

## Features

- Ultra modern dark theme with neon accents
- Glassmorphism effects
- Smooth scroll animations with Framer Motion
- Fully responsive design
- WhatsApp integration
- 10 sections: Hero, About, Programs, Pricing, Trainers, Gallery, Testimonials, CTA, Contact, Footer

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Deploy on Vercel

1. Push to GitHub
2. Connect to Vercel
3. Deploy!
